# Plan: Scraper Enhancements

**Date:** 2026-04-06
**Branch:** feat/scraper

## Goal
Make rdt-cli-go useful as a no-login scraper: auto-paginate all listing
commands and export results to CSV/JSON files.

## Scope
- `--all` flag: auto-paginate until exhausted, on every listing command
- `--output <file>`: write results to .csv or .json (auto-detect by extension)
- No auth features. No social commands.

## Success Criteria
1. `rdt browse user-posts kn0thing --all --limit 100` returns >25 items
2. `--output out.json` produces valid JSON file
3. `--output out.csv` produces CSV with headers
4. `--all --json` outputs valid envelope with all items
5. Commands without `--all` behave identically to before
